### Endpoint

El endpoint para interactuar con la pagina de libros es

`/books`

Donde se encontrará disponible un botón para crear libros (hard coded), ver o editar libros dirigiendote a un nuevo url y al completar las operaciones se redirige nuevamente al `/books`.

Se le agregó un modal sin mucho detalle para notificar de errores en el Index (/books).
